## 稳定性与弱样本

- `temp25_lambda5_outer60` 在 50/50 个样本上 IoU 都高于 baseline。
- 所有 CSV 指标均为 finite；runner stderr 为空。
- `temp25_lambda5_outer60` 的低 IoU 样本主要包括 sample 40、12、34、21、36、0、9、10。
- 最明显弱样本是 sample 40：`defect_iou=9.043478e-02`，`defect_area_pred=8625`，`defect_area_label=780`，表现为明显 false positive / 面积扩张。
- 其他弱样本多数仍优于 baseline，但存在面积偏大或定位/形状细节不足。

## 当前结论

`temp25_lambda5_outer60` 在新的 200x100 / 50-sample 数据上稳定优于 baseline，平均 IoU、面积控制和连续 `mu` 误差均显著改善。因此它可以作为当前 200x100 半监督 runner 的默认候选。

该结果仍然是 BCE mask prior 使用 `mu_label < 500` 的半监督 / 诊断上界，不是无监督 weak-form 反演成功。后续若继续推进，建议优先做 200x100 弱样本可视化与 failure taxonomy，而不是继续盲目扫描 radius / centers / area prior。
