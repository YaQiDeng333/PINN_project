## 稳定性与弱样本

- `temp25_lambda5_outer60` 在 100/100 个样本上 IoU 都高于 baseline。
- 所有 CSV 指标均为 finite；两组 runner stderr 为空。
- `temp25_lambda5_outer60` 的平均 IoU 为 `7.595984e-01`，仍显著高于 baseline 的 `7.472148e-02`。
- 明显弱样本主要包括 sample 40、47、17、57、73、95、54、29、49、43、19、86。
- 最弱样本 sample 40：`defect_iou=1.322563e-01`，`defect_area_pred=8975`，`defect_area_label=1187`，主要表现为 false positive / 面积扩张。

## 当前结论

`temp25_lambda5_outer60` 在新的 200x100 / 100-sample 数据上稳定优于 baseline，并且在 IoU、面积控制、`mu_mse` 和 `mu_mae` 上都有数量级改善。因此它可以继续作为当前 200x100 半监督 runner 的默认候选。

该结果仍然是 BCE mask prior 使用 `mu_label < 500` 的半监督 / 诊断上界，不是无监督 weak-form 反演成功。后续若继续推进，建议围绕 200x100 弱样本做可视化和失败类型分类，而不是继续盲目扫描 radius / centers / area prior。
