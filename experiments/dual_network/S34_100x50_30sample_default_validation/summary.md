# S34 100x50 30-Sample Default Validation

## 数据生成

命令：

```powershell
python data_generator_v2.py --train-samples 30 --val-samples 0 --test-samples 0 --grid-x 100 --grid-y 50 --output-dir experiments/dual_network/S34_100x50_30sample_default_validation/data --seed 1034
```

使用的 `.npz`：

- `experiments/dual_network/S34_100x50_30sample_default_validation/data/training_data_train.npz`

## 实验配置

- 网格分辨率：100x50
- 样本数：30 train samples
- 训练步数：`outer_steps=30`，`phi_steps=30`，`mu_steps=30`
- 共同配置：`center_mode=three`，`test_radius=5.0`，`lambda_area_prior=1.0`，`lambda_mask_prior=1.0`
- baseline：`lambda_mask_bce_prior=0.0`，`mask_prior_temperature=50.0`
- `temp25_lambda3`：`lambda_mask_bce_prior=3.0`，`mask_prior_temperature=25.0`

## 当前结论

`temp25_lambda3` 在 100x50 / 30 samples 下仍明显优于 baseline，说明半监督 BCE 上界在中高分辨率下仍有效。相比 S28 80x40，100x50 的 baseline 低 `mu` 扩散更严重，`temp25_lambda3` 的平均 IoU 仍保持较高水平，但 `defect_area_pred` 和连续 `mu` 误差随分辨率上升仍需要继续关注。

该结果仍属于半监督 / 诊断上界，因为 BCE mask prior 使用 `mu_label < 500`；不能表述为无监督 weak-form 成功。
