# S205 COMSOL V3 / hard-case data request summary

S205 drafts `COMSOL_V3_HARD_CASE_DATA_REQUEST.md` as a small targeted hard-case pack request. It is not a broad data expansion and does not claim training improvement.

The request preserves the V2 schema and focuses on x-bin boundary cases, slot 0 / slot 2 fragility, near/medium component spacing, mixed type and rotated-rect cases, and all-bins-correct-but-low-IoU samples where decoded center / offset / geometry interaction may dominate.

Suggested size is `20/5/5` as a pilot, or `40/10/10` if generation budget allows. Acceptance is converter/schema/metadata readiness, not model performance.
