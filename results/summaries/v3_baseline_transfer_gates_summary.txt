# 第 9.1 / 9.2 baseline transfer gates summary

## 实验说明

本轮只做 fast gate，用于判断 v4 阶段中看起来有价值的 loss / auxiliary 技巧是否能迁移到当前 v3_complex baseline。

当前 baseline 不变，不扩展 3 seed，不继续调 `lambda_mask`、`lambda_dice`、`lambda_area`，不继续把 v4 的 loss / aux / threshold 技巧迁移到 v3_complex baseline。

## CURRENT_BASELINE

模型：

`checkpoints/best_model_v3_complex_tv_sweep_2e-6.pt`

数据集：

* train: `data/training_data_v3_complex_train.npz`
* val: `data/training_data_v3_complex_val.npz`
* test: `data/training_data_v3_complex_test.npz`

配置：

* `lambda_tv = 2e-6`
* physics_loss: off
* L-BFGS: off

test 指标：

| metric | value |
|---|---:|
| MSE | 2.07377174e+04 |
| MAE | 4.44655262e+01 |
| IoU | 2.95272047e-01 |
| Dice | 4.21885407e-01 |
| area_error | 3.94517442e-01 |
| center_error | 1.32594189e+00 |

## 第 9.1：aux mask regularizer transfer gate

模型：

`checkpoints/best_model_v3_complex_aux_mask_seed42_tv2e-6.pt`

配置：

* seed = 42
* dataset = `v3_complex`
* decoder = standard
* loss_type = `mse`
* `lambda_tv = 2e-6`
* `--aux-mask-head`
* `lambda_mask = 0.1`
* final mask source = `pred_mu < 500`

test 指标：

| metric | value |
|---|---:|
| MSE | 2.08002269e+04 |
| MAE | 4.69429331e+01 |
| IoU | 2.75571389e-01 |
| Dice | 3.95236365e-01 |
| area_error | 4.19878977e-01 |
| center_error | 1.26998199e+00 |

失败原因：

除 `center_error` 改善外，MSE、MAE、IoU、Dice、area_error 均比 CURRENT_BASELINE 变差。该 gate 没有明确综合改善信号，不扩展 3 seed。

## 第 9.2：shape-aware loss transfer gate

模型：

`checkpoints/best_model_v3_complex_shape_aware_seed42_tv2e-6.pt`

配置：

* seed = 42
* dataset = `v3_complex`
* decoder = standard
* loss_type = `weighted_mse_dice_area`
* `defect_weight = 5`
* `lambda_dice = 0.03`
* `lambda_area = 0.04`
* `lambda_tv = 2e-6`
* aux_mask_head: off
* final mask source = `pred_mu < 500`

test 指标：

| metric | value |
|---|---:|
| MSE | 2.89295339e+04 |
| MAE | 6.98343477e+01 |
| IoU | 3.69288142e-01 |
| Dice | 5.24763034e-01 |
| area_error | 9.16736588e-01 |
| center_error | 1.12094815e+00 |

失败原因：

IoU、Dice、center_error 改善，但 MSE、MAE、area_error 明显变差，尤其 area_error 从 0.3945 升到 0.9167。该 gate 触发停止条件，不扩展 3 seed。

## 最终结论

第 9.1 aux mask regularizer transfer gate 失败。

第 9.2 shape-aware loss transfer gate 失败。

两者都不能挑战 CURRENT_BASELINE。CURRENT_BASELINE 不变，仍为：

`checkpoints/best_model_v3_complex_tv_sweep_2e-6.pt`

baseline transfer gates 停止。不继续调 `lambda_mask`、`lambda_dice`、`lambda_area`，也不继续把 v4 的 loss / aux / threshold 技巧硬迁移到 v3_complex baseline。

v4 / aux / threshold / shape-aware transfer 线到此停止。后续主线转向 CURRENT_BASELINE failure-driven analysis。
