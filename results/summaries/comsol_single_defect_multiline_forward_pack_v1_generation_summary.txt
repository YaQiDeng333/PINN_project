## 3. 当前数据与目标 pack 的缺口

目标 pack `comsol_single_defect_multiline_forward_pack_v1` 至少需要：

- multi-line `delta_Bz`，即 shape 类似 `(N, n_lines, L)`；
- 2D / quasi-2D defect mask，shape 类似 `(N, H, W)`；
- `mask_x` / `mask_y` 坐标；
- `sensor_x` / `scan_line_y` / sensor height；
- mask 与 COMSOL 几何坐标的一致映射；
- single-defect 类型标签，最好包含 `polygon` / `rotated_rect` 或可明确映射的 rectangular notch；
- smoke split 或明确的 `split=smoke` 标记；
- 生成脚本和 inspect 脚本能重复构建同一 schema。

当前已有 5 个 rectangular notch 样本仍缺：

- multi-line：当前 `sensor_y` 只有单一值 `0.0`；
- 2D / quasi-2D mask：没有 `mask` / `masks` / `mu_maps` 字段；
- mask 坐标映射：没有 `mask_x` / `mask_y`；
- polygon / rotated_rect 标签：只有 `rectangular_notch`；
- train / val / test 或 smoke split：没有正式 split；
- 当前 `prepare_comsol_mfl_npz.py` 只整理已有单线 CSV，不生成 mask，也不生成 multi-line pack。

因此当前数据不能直接声明为 train-ready。

## 6. 如果要生成 v1 pack，最小输入要求

下一步需要一个可复现的 COMSOL / MCP 生成入口，输出至少以下 schema：

### 每个样本

- `sample_id`
- `defect_type`
- `geometry_params`
  - rectangular / rotated_rect: `center_x`, `center_y`, `width`, `height_or_length`, `depth`, `angle` 如果支持
  - polygon: `vertices` 如果支持
- `material`
- `sensor_liftoff`
- `scan_line_y`
- `sensor_x`
- `Bz_defect`
- `Bz_reference` 或 `Bz_no_defect`
- `delta_Bz = Bz_defect - Bz_reference`
- `mask`
- `mask_x`
- `mask_y`
- `coordinate_units`
- `simulation_settings`
- `split` 或 `split=smoke`

### 建议 smoke 规模

- 最小 6 个样本；
- 推荐 12 个样本；
- 本阶段不超过 20 个样本；
- 只做 single-defect，不做 multi_defect；
- 优先 rectangular_notch / rotated_rect；
- polygon 如果生成脚本稳定后再加入。

## 7. 推荐下一步

推荐下一步不是训练，而是先补齐可复现生成链路：

1. 在 COMSOL_Multiphysics_MCP 侧扩展或新增一个 explicit multi-line generator：
   - 支持 `scan_line_y = [-y0, 0, +y0]` 或更多固定线；
   - 同时输出 reference 与 defect 的 Bz；
   - 保留 `.mph`、CSV、summary 和 manifest；
   - 记录 COMSOL 几何坐标与 mask 坐标。

2. 在 PINN_project 侧新增准备脚本：
   - 只读取生成结果；
   - rasterize 同一 geometry 得到 2D / quasi-2D mask；
   - 写出 `data/comsol_mfl/prepared/comsol_single_defect_multiline_forward_pack_v1.npz`；
   - 写出 inventory CSV；
   - 明确 `signal_kind=delta_Bz` 或 `raw_Bz`，不得混用。

3. 先运行小规模 inspect：
   - 样本数；
   - signals shape；
   - masks shape；
   - NaN / inf；
   - mask area；
   - signal 是否非零；
   - 是否真 multi-line；
   - 是否真 delta_Bz；
   - 坐标字段是否齐全。
