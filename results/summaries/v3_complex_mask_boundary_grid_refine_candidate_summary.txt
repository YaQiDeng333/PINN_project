# v3_complex mask-only grid decoder + coordinate refinement candidate

This RESULT_DRIVEN_EXPERIMENT trains an independent mask-only grid decoder + coordinate refinement candidate for seeds 42, 123, and 2026. It does not modify train_pinn.py, evaluate_pinn.py, data_generator_v2.py, CURRENT_BASELINE.md, README.md, EXPERIMENT_LOG.md, the route document, or NEXT_STEP.md.

## Model and loss

The model keeps the direct Bz -> mask setup. It reuses BzEncoder, projects the latent vector into a low-resolution 2D feature map, upsamples it with ConvTranspose2d blocks, then concatenates normalized coordinate channels and fixed Fourier coordinate channels before a lightweight convolution refinement head outputs full-grid mask logits.

* BzEncoder latent dimension: 64
* low-resolution feature map: 64 x 10 x 20
* coordinate channels: x, y, and fixed Fourier frequencies 1, 2, 4, 8
* refinement head: Conv2d blocks over grid features + coordinate features

Training loss is BCEWithLogits + soft Dice. No SDF, boundary head/loss, loss-weight sweep, adaptive threshold, post-processing, or ensemble is used.

Train mask positive fraction: 0.036335. BCE pos_weight uses sqrt(neg/pos) capped at 8.0; value used: 5.149934.

## Closure decision

Grid + coordinate refinement improves overall IoU/Dice and center_error relative to the current mask-only grid decoder CURRENT_BASELINE. Small, low-signal, polygon, and rotated_rect IoU/Dice also improve, and pred_area=0 decreases. However, area_error is clearly worse overall and in the key small / low-signal / polygon / rotated_rect groups. Visual preview inspection still shows round-blob / misaligned polygon behavior rather than a clear fine-boundary fit.

The candidate does not satisfy the full acceptance criteria. It is not promoted to a new CURRENT_BASELINE. No refinement v2, Fourier-feature count tuning, or refinement-head structure changes are continued. CURRENT_BASELINE remains the mask-only grid decoder boundary model with validation-selected threshold=0.90.

Preview PNGs are written to `results\previews\mask_boundary_grid_refine_candidate` for visual inspection of boundary shape.
