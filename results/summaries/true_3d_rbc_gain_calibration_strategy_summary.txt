20.89 gain/amplitude calibration-only strategy audit

Scope:
- Dataset: comsol_true_3d_rbc_imported_watertight_pilot_v3_240
- Model: fixed 20.77/20.85 seed=42 baseline checkpoint from 20.88a artifact manifest
- Operation: perturb delta_b in memory, then apply input calibration in memory
- No COMSOL, no NPZ/data write, no training, no CURRENT_BASELINE update

Calibration strategies:
- no_calibration: raw perturbed delta_b
- per_sample_rms_normalization: scale each sample to train median RMS
- per_axis_rms_train_stats: scale Bx/By/Bz separately to train median axis RMS
- peak_amplitude_normalization: scale each sample to train median absolute peak
- robust_median_iqr_scale: match each sample robust median/IQR to train robust scale
- reference_proxy_global_gain_correction: delta_b RMS proxy because independent no-defect scale is not available

No-calibration reference on test split:
- clean profile_depth_rmse_m: 0.000387737
- gain 0.8 degradation vs 20.85: 123.845%
- gain 1.2 degradation vs 20.85: 69.657%
- Bx 50pct attenuation degradation vs 20.85: 141.577%

Validation-selected calibration candidate:
- strategy: per_axis_rms_train_stats
- validation selection score: 95.742170
- test clean profile RMSE drop vs 20.85: 21.194%
- test gain 0.8 profile degradation: 21.194%
- test gain 0.8 degradation reduction vs no calibration: 82.886%
- test gain 1.2 profile degradation: 21.194%
- test gain 1.2 degradation reduction vs no calibration: 69.573%
- test clean Dice: 0.840128
- test Bx 50pct attenuation degradation: 12.331%

Calibration-only sufficient: False
Decision reason: calibration_only_not_enough_gain_sensitivity_remains_or_clean_drop_too_large

Interpretation:
- Calibration strategy selection uses validation split only; test split is final reporting.
- Calibration-only is considered sufficient only if test clean profile RMSE remains within +10%, gain 0.8 and 1.2 degradation are reduced by at least 50%, and clean Dice stays within 0.02 of baseline.
- If this fails, Stage C augmentation training gate should run because the model remains gain/amplitude sensitive.
