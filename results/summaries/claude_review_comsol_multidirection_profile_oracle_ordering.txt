### 建议修复

1. **缺少 `direction_0_Bz_trainstd_normalized` 对照。** `direction_0_Bz_only` 使用 per-sample NRMSE（逐样本 `np.std(ref)` 归一化），而 `multi_direction_Bz_trainstd_normalized` 使用固定 train-only 方向-轴尺度。test delta +0.1091（0.4545 → 0.5636）无法排除归一化策略差异的混杂。但是 `direction_90_Bz_only`（同样使用 NRMSE）相比 `direction_0_Bz_only` 也改善了 +0.0727，说明 excitation direction change 本身确实产生了信号 — 只是与 multi-direction Bz 的差异可能被归一化放大。添加 `direction_0_Bz_trainstd_normalized` 和 `direction_90_Bz_trainstd_normalized` 作为对照可完全隔离此混杂。

2. **test split 仅 2 base samples（16 rows、55 pairs）。** 排序准确率估计方差大。per-source_defect_type test 仅 1 base（各 8 rows），无法按 rect/rot 分组可靠报告。当前 multi_direction_Bz test 0.5636 是此系列首次正 delta，但统计功效低。若后续验证 Bz-only 信号，至少需要 4-6 test bases。

### 是否认为 multi-direction excitation 有助于解决 profile residual non-identifiability

**部分有助于，但不充分。** 关键证据：
- test multi_direction_Bz delta = **+0.1091**（0.4545 → 0.5636），是 20.61–20.64 四个 stage 中**首次正 delta**
- test direction_90_Bz_only（NRMSE）相比 direction_0_Bz_only 改善 +0.0727（0.4545 → 0.5273），**同一归一化策略下** excitation direction change 本身产生了真实信号改善
- 但绝对值 0.5636 < 0.65 gate，val delta 为负（-0.0179），correlation 为强负（-0.737），信号方向正确但幅度和稳定性不足
- multi_direction_all_axis 完全退化（0.3455），Bx/By 在 multi-direction 下引入噪声

---

### 是否建议下一步训练 multi-direction profile surrogate

**不建议。** Gate 未通过（test_multi_all = 0.3455 < 0.65，correlation = -0.803）。即使 Bz-only 正信号 +0.1091 是突破性的（系列首次正 delta），val delta 为负和强负 correlation 表明 cross-validation 不稳定，训练 surrogate 极可能重复 20.55/20.57/20.59 的 mismatch。

---

### 下一步更建议什么

**Option C（true 3D profile / Piao-style route）** 是当前 route decision 的推荐且合理。同时可考虑 **Option B 变体**：若资源允许，扩 multi-direction Bz-only pack（更多 test bases、更多角度如 30°/60°）验证 +0.1091 正信号是否稳定，因为这个方向上的首度突破不应被忽略。

### 简短原因

COMSOL 真实 multi-direction excitation（Je 正交旋转）在 test Bz-only 上产生了 20.61–20.64 系列**首次正向 oracle ordering delta**（+0.1091），且 direction_90 vs direction_0 在**同一 NRMSE 归一化下**仍有 +0.0727 改善，排除归一化混杂后仍证明激励方向变化比增加同方向场分量（20.63 multi-axis）或改变 liftoff（20.62 multi-height）更有信号价值。但绝对值 0.5636 离 0.65 gate 有差距，val delta 为负，correlation 为强负，信号不够稳定以支撑 surrogate 训练。当前 2 test bases 统计功效低。建议要么转向 true 3D Piao-style 路线，要么扩 multi-direction Bz-only pack 充分验证正信号。
