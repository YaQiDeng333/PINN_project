# v3_complex mask-only grid decoder + boundary head candidate

This RESULT_DRIVEN_EXPERIMENT trains an independent mask-only grid decoder + boundary head candidate for seeds 42, 123, and 2026. It does not modify train_pinn.py, evaluate_pinn.py, data_generator_v2.py, CURRENT_BASELINE.md, README.md, EXPERIMENT_LOG.md, the route document, or NEXT_STEP.md.

## Model and loss

The model keeps the direct Bz -> mask setup. It reuses BzEncoder, projects the latent vector into a low-resolution 2D feature map, upsamples it with ConvTranspose2d blocks, then uses two heads:

* BzEncoder latent dimension: 64
* low-resolution feature map: 64 x 10 x 20
* mask_logit head for defect probability
* boundary_logit head for a 1-pixel morphological boundary band

The boundary target is generated from `target_mu_norm < 0.5` masks with a torch max-pool dilation/erosion boundary band. Training loss is mask BCEWithLogits + soft Dice + 0.10 * boundary BCEWithLogits. No SDF, loss-weight sweep, adaptive threshold, post-processing, or ensemble is used.

Train mask positive fraction: 0.036335. BCE pos_weight uses sqrt(neg/pos) capped at 8.0; value used: 5.149934.
Train boundary positive fraction: 0.015744. Boundary BCE pos_weight uses sqrt(neg/pos) capped at 8.0; value used: 7.906697.
