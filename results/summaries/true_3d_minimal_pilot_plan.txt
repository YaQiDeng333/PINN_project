Stage 20.67 pilot, preliminary design:
- Sample count: 120 samples minimum, split 80/20/20.
- If COMSOL runtime is acceptable, expand to 240 samples, split 160/40/40.
- Single-defect only.
- Shape coverage: flat rectangular-like, smooth RBC/ellipsoid-like, rotated rectangular-like.
- Parameter ranges: L 4-14 mm, W 2-8 mm, D 0.4-2.0 mm, angle 0-75 deg where applicable.
- Observation ablations: Bz @ 0.008, BxByBz @ 0.008, Bz @ 0.008+0.012, BxByBz @ 0.008+0.012.

Preliminary acceptance guidance for 20.67:
- These are provisional targets, not validated hard standards.
- projected mask IoU >= 0.65.
- Dice >= 0.78.
- normalized profile error <= 0.25.
- D MAE <= 0.30 mm.
- Main richer observation candidate improves over Bz @ 0.008 by IoU >= +0.04 or normalized profile error <= -0.05.

Stop conditions:
- variable-depth COMSOL solid cannot be constructed.
- projected mask, depth map, and COMSOL solid are not same-source.
- delta_B check fails.
- true-geometry forward residual shows systematic peak shift.
- future oracle residual cannot exceed the weak 20.64 signal.
