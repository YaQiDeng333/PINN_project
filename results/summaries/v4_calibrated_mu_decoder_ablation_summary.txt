# 第 7.20B calibrated_mu 轻量 decoder 增强实验总结

## 实验目标

在第 7.20A `calibrated_mu` 输出参数化基础上，只增强 decoder 容量，验证 decoder 表达能力是否限制 μ 校准和 area_error 改善。

## 固定配置

- dataset = v4_balanced_complex
- seed = 42
- model_variant = calibrated_mu
- loss_type = weighted_mse_dice_area
- defect_weight = 5
- lambda_dice = 0.03
- lambda_area = 0.04
- area_loss_type = symmetric
- lambda_tv = 0
- physics_loss = off
- L-BFGS = off
- epochs = 100

## A/B 模型

### standard decoder

- checkpoint = checkpoints/best_model_v4_calibrated_mu_seed42_w5_dice003_area004.pt
- decoder = 128 / 128 / 64 + Tanh
- 说明：复用第 7.20A 的 calibrated_mu + standard decoder seed=42 checkpoint。
- 注意：第 7.20B 对比中的 standard decoder 结果不是本轮重新训练结果。

### enhanced decoder

- checkpoint = checkpoints/best_model_v4_calibrated_mu_enhanced_decoder_seed42_w5_dice003_area004.pt
- decoder = 256 / 256 / 128 / 64 + SiLU
- BzEncoder、Fourier feature、calibrated_mu 输出映射均保持不变。

## 结论

enhanced decoder 能进一步降低缺陷区预测 μ_r，defect_mu_mean 从约 361 降到约 333，defect_mu_median 从约 262 降到约 238，说明 decoder 容量确实会影响 μ 校准。

enhanced decoder 也小幅改善 Dice、MAE、center_error、small polygon IoU / Dice、small polygon IoU=0 数量和 multi_defect center_error，并保持 small polygon `pred_area=0` 为 0 / 25。

但 enhanced decoder 明显放大了面积高估：overall area_error 从 0.6401 升到 0.9582，polygon area_error 从 0.7938 升到 1.4199，medium polygon area_error 从 1.0484 升到 1.6387，pred_area > true_area 从 182 / 200 增加到 189 / 200。因此本轮不能说明 enhanced decoder 是更优整体方案，也不切换 CURRENT_BASELINE。

本轮结论是：decoder 表达能力是 μ 校准的影响因素之一，但当前 enhanced decoder 与现有 loss 配置配合后会加重面积高估。若继续推进结构实验，建议先做 seed repeat 验证该 trade-off 是否稳定，再考虑更温和的 decoder 或面积校准策略。
