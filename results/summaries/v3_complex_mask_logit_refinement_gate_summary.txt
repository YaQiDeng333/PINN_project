## Current baseline coarse prediction

CURRENT_BASELINE is loaded from the three mask-only grid decoder + forward consistency checkpoints recorded in scripts.train_geometry_forward_consistency_candidate.CURRENT_BASELINE_CHECKPOINTS:

* `checkpoints/mask_boundary_forward_consistency_lambda_bracket/best_forward_consistency_lambda_0p10_seed42.pt`
* `checkpoints/mask_boundary_forward_consistency_lambda010/best_mask_boundary_forward_consistency_seed123.pt`
* `checkpoints/mask_boundary_forward_consistency_lambda010/best_mask_boundary_forward_consistency_seed2026.pt`

The script computes each seed probability map and uses the 3-seed mean probability as `coarse_prob`. The fixed CURRENT_BASELINE threshold is `0.8`. Bz residual is computed with the frozen mask-to-Bz surrogate loaded via `load_forward_surrogate`.

## Refinement setup

For each validation/test sample, `coarse_prob` is clamped and converted to logits. Only those logits are optimized for 50 Adam steps at lr=0.03. No true mask is used during optimization.

* proposal_only: MSE(refined_prob, coarse_prob) + 0.001 * TV(refined_prob)
* proposal_forward: MSE(refined_prob, coarse_prob) + 0.1 * MSE(frozen_surrogate(refined_prob), Bz_obs) + 0.001 * TV(refined_prob)

No threshold sweep, post-processing, weight tuning, or refinement v2 is used.
