# v3_complex mask-only grid decoder candidate

This RESULT_DRIVEN_EXPERIMENT trains an independent mask-only grid decoder candidate for seeds 42, 123, and 2026. It does not modify train_pinn.py, evaluate_pinn.py, data_generator_v2.py, CURRENT_BASELINE.md, README.md, EXPERIMENT_LOG.md, the route document, or NEXT_STEP.md.

## Model and loss

The model keeps the direct Bz -> mask setup. It reuses BzEncoder, projects the latent vector into a low-resolution 2D feature map, then upsamples it with ConvTranspose2d blocks to produce full-grid mask logits.

* BzEncoder latent dimension: 64
* low-resolution feature map: 64 x 10 x 20
* decoder: ConvTranspose2d upsampling blocks followed by 2D convolution mask-logit head

Training loss is BCEWithLogits + soft Dice. No SDF/boundary loss, loss-weight sweep, adaptive threshold, post-processing, or ensemble is used.

Train mask positive fraction: 0.036335. BCE pos_weight uses sqrt(neg/pos) capped at 8.0; value used: 5.149934.
