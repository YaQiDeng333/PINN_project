True 3D pilot dataset schema design

Schema version:
- true3d_profile_v1_piao_rbc

Observation fields:
- delta_b_multi_axis: (N, A, L, X), float32, A=3 axes, L=scan lines, X=sensor_x points.
- b_defect_multi_axis: same shape as delta_b_multi_axis.
- b_no_defect_multi_axis: same shape as delta_b_multi_axis.
- axis_names: ["Bx", "By", "Bz"].
- axis_expressions: ["mf.Bx", "mf.By", "mf.Bz"].
- sensor_x: (X,), float32.
- scan_line_y: (L,), float32.
- sensor_z_m: scalar or (Z,), first smoke fixed at 0.008.

Primary label fields:
- rbc_params: (N, 1, 6), float32.
- rbc_param_names: ["L", "W", "D", "wLD", "wWD", "wLW"].
- profile_pose: (N, 1, 5), float32 with center_x_m, center_y_m, angle_rad, length_m, width_m.
- profile_depth_max_m: (N, 1), float32.
- component_count: (N,), int32, pilot fixed to 1.
- component_valid: (N, 1), bool.

Derived 3D/profile fields:
- profile_u: (Ku,), float32.
- profile_v: (Kv,), float32.
- profile_depth_grid_m: (N, 1, Ku, Kv), float32.
- profile_depth_map_xy_m: (N, H, W), float32.
- profile_footprint_mask: (N, H, W), bool or uint8.
- profile_z_reference_m: scalar or (N,), float32.

2D compatibility fields:
- masks: (N, H, W), bool or uint8. Same as profile_footprint_mask for generated true 3D samples unless an external mask comparator is present.
- mask_x: (W,), float32.
- mask_y: (H,), float32.
- quality_to_2d_mask_json: (N,), string.

Metadata fields:
- sample_ids: (N,), string.
- split: (N,), string.
- defect_types: (N,), string.
- geometry_params_json: (N,), string.
- source_metadata_json: (N,), string.
- schema_version: scalar string.

Projection rule:
- RBC params + pose -> depth(x,y).
- footprint = depth(x,y) > max(1e-6 m, 0.01 * D).
- profile_footprint_mask is sampled on mask_x/mask_y.
- projected mask metrics compare this footprint to any legacy 2D mask comparator, but depth correctness must be judged by depth/profile metrics.

20.66 smoke observation:
- Bx/By/Bz at sensor_z_m=0.008 only.
- 0.012 m is reserved for 20.67 pilot schema/future ablation and is not a 20.66 smoke requirement.
