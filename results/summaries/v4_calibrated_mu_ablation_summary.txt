# 第 7.20A calibrated_mu 输出参数化校准实验总结

## 实验目标

验证只改变输出 μ 参数化、保持 BzEncoder 和 decoder 主体不变时，是否能缓解 v4_balanced_complex 模型的 μ 值偏软和预测面积偏大问题。

## 固定配置

- dataset = v4_balanced_complex
- seed = 42
- loss_type = weighted_mse_dice_area
- defect_weight = 5
- lambda_dice = 0.03
- lambda_area = 0.04
- area_loss_type = symmetric
- lambda_tv = 0
- physics_loss = off
- L-BFGS = off
- epochs = 100

## A/B 模型

### baseline

- checkpoint = checkpoints/best_model_v4_baseline_seed42_w5_dice003_area004.pt
- output = Linear(64, 1) + Softplus

### calibrated_mu

- checkpoint = checkpoints/best_model_v4_calibrated_mu_seed42_w5_dice003_area004.pt
- output = sigmoid(logit) 得到 defect_prob，再映射到 mu_norm ∈ [0.001, 1.0]
- 映射方式：mu_pred = 0.001 + (1.0 - 0.001) * (1 - defect_prob)

## 结论

calibrated_mu 相比 baseline seed=42 小幅改善 MSE、IoU、Dice、center_error、polygon area_error、small polygon IoU / Dice、medium polygon area_error 和 multi_defect center_error；small polygon 仍保持 0 / 25 漏检。需要注意：small polygon pred_area=0 为 0 / 25 只说明没有完全空预测，仍需要关注 small polygon IoU=0 的样本；后续评估应同时记录 small polygon IoU=0 数量。

calibrated_mu 的缺陷区预测 μ_r 分布有所降低，defect_mu_mean 从约 399 降到约 361，defect_mu_median 从约 295 降到约 262，说明输出参数化校准方向有效但幅度有限，缺陷区仍远未接近真实 μ_r≈1。

area_error 只从 0.6404435 降到 0.6401099，改善很小；pred_area > true_area 数量反而从 174 / 200 增加到 182 / 200。因此 calibrated_mu 不能单独解决预测面积系统性偏大的问题。

本轮不切换 CURRENT_BASELINE。第 7.20A 结果支持继续做第 7.20B decoder 轻量增强实验，但需要保持 seed=42 和同一 loss 配置，以便确认 decoder 表达能力是否能进一步改善 μ 校准和面积误差。
