Piao 2019 revised POC scope summary

Stage
- 20.47-revised: Piao-2019-inspired Bz-only NLS-style geometry parameter inversion POC.

Paper alignment
- Piao et al. 2019 uses a Rational Bezier Curve (RBC) model to generate 3D defect profiles from six geometric parameters: L, W, D, wLD, wWD, wLW.
- Piao et al. 2019 uses nonlinear least squares (NLS) to fit exponential models to three-axis MFL signals Bx, By, Bz, first along the axial direction and then along the tangential direction, yielding 18 key physics-based feature parameters.
- Piao et al. 2019 uses RBF-kernel LS-SVM to map those 18 feature parameters to the six RBC geometry parameters.

Current revised POC position
- This is not a Piao 2019 full reproduction.
- This is a weak Piao-2019-inspired Bz-only 2D/quasi-2D engineering adaptation.
- The current project input is multi-line delta_bz only, not tri-axis Bx / By / Bz matrices.
- The current project target is 2D/quasi-2D geometry parameters from geometry_params, not a 3D RBC profile.
- RBC / tri-axis NLS / LS-SVM are not fully implemented.
