# DUAL_NETWORK_EXPERIMENT_LOG

## 0. Branch Position

This log records the `feature/dual-network-variational` branch.

The branch explores a dual-network variational inversion path:

- `phi-Net` reconstructs a scalar field `phi(x, y)`;
- `mu-Net` predicts a permeability field `mu(x, y)`;
- `phi` is trained through field/data terms;
- `mu` is updated through weak-form material residuals and optional priors.

This branch does not replace `main`. The strongest current results use BCE mask supervision from `mu_label < 500`, so they are semi-supervised / diagnostic upper-bound results, not proof of unsupervised weak-form inversion success.
